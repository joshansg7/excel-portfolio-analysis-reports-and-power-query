This repo contains some practical Power Query workflows I built while learning data analysis concepts in Microsoft Excel.

Things covered include:

importing CSV files
cleaning messy data
removing duplicates/nulls
merge queries (joins)
append queries
folder-based refresh workflows

The aim was to understand how analysts automate repetitive reporting tasks using reusable Power Query transformations.

Most of the repo is documented through screenshots and a short PDF walkthrough showing the different workflows and outputs.

The repo also contains some example charts that analyse example datasets
